## What is the VGS - Toolkit ?

The Victor Game Studio - Toolkit is a set of tools to enable awesome moding support for the games made by [Victor Game Studio](https://victorgamestudio.com). It includes a lua scripting engine made to create new gameplay elements, UI, and much more!
